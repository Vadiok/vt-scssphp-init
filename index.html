Колонки одинаковой высоты (Equal Height Blocks)

jQuery плагин для установки одинаковой высоты дочерним элементам своего родителя. Параметры можно посмотреть в modules/equal-height-blocks/equal-height-blocks.js.

Плагин используется на этой странице для колонок Bootstrap выше, для проверки можно изменять размер окна, чтобы колонки перестраивались - их высота будет пересчитываться.
